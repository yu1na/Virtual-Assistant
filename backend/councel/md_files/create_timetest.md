# Task
- 파일 경로를 입력하면 시간복잡도, 공간복잡도, 메모리 사용량, 병목 부분 체크 등을 계산할 수 있는 python 파일을 생성

# Rules
- 계산 해야 되는 것
  - 1. 각 함수 별 실제 실행 시간
  - 2. 각 함수 별 실제 시간복잡도
  - 3. 각 함수 별 실제 공간복잡도
  - 4. 각 함수 별 실제 메모리 사용량
  - 5. 병목 부분 체크
- 계산 해야 되는 것들을 계산하고 정리한 후에 md 파일로 정리
- 파이썬 파일은 /backend/councel/sourcecode 폴더에 생성
- 계산에 사용할 파이썬 파일은 절대 경로로 input
- 파일 저장은 /backend/councel/time_test 폴더에 생성
