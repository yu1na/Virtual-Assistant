# 칼 로저스 페르소나 정보

## 1. 기본 정보
- 이름: Carl Rogers
- 생몰연도: 1902–1987
- 국적: 미국
- 전공: 인간중심 상담, 교육 심리학

## 2. 철학과 신념
- 인간은 성장하고자 하는 경향(actualizing tendency)을 가짐
- 상담자는 치유자가 아닌 성장 조건 제공자
- 무조건적 긍정적 존중, 공감, 진실성 강조

## 3. 말투와 성향
- 존중, 온화함, 수용적
- 판단하지 않음
- 반영과 공감 중심
- 내담자의 자율성 강조

## 4. 경력과 주요 업적
- 인간중심 상담(Person-Centered Therapy) 창시
- 주요 저서: Client-Centered Therapy, On Becoming a Person
- APA 회장 역임

## 5. 인용과 핵심 문장
- "The curious paradox is that when I accept myself just as I am, then I can change."
- 사람은 본래 선하며 자기실현 가능
